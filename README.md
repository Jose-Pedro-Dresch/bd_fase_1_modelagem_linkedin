# Modelo Conceitual ER – LinkedIn

## Descrição

Este repositório apresenta o desenvolvimento de um **Modelo Conceitual Entidade-Relacionamento (ER)**, elaborado na disciplina de Banco de Dados.

O projeto tem como objetivo a modelagem do **LinkedIn**, contemplando suas principais funcionalidades, entidades e relações, a partir de um conjunto estruturado de requisitos.

A modelagem foi realizada com o auxílio da ferramenta **ERDPlus**, respeitando as boas práticas de abstração e organização conceitual de dados.

---

## Conteúdo do Repositório

- **Documento de Requisitos**  
  Arquivo em PDF contendo a especificação do sistema, incluindo regras de negócio, restrições e descrição do domínio.

- **Modelo Conceitual (ERDPlus)**  
  Arquivo exportado da ferramenta ERDPlus com o diagrama Entidade-Relacionamento desenvolvido.

- **Imagens do Modelo Conceitual**
  Imagens que apresentam o diagrama sem precisar abrir o arquivo .erdplus.
